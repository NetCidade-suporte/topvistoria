
# Directive: Example Task

## Goal
Demonstrate how to write a directive.

## Inputs
- `input_text`: The text to process.

## Tools
- `execution/example_script.py`

## Instructions
1. Run `execution/example_script.py` with the `input_text`.
2. Save the output to `.tmp/output.txt`.
3. Read the output and present it to the user.

## Edge Cases
- If `input_text` is empty, return an error message.
